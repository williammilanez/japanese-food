## Sobre o projeto 📝

Este é um projeto de site simples para um restaurante de **comida japonesa**, desenvolvido com foco em simplicidade, responsividade e funcionalidade direta com o WhatsApp.

## Como Usar 🚀

- Clique no botão "Fazer meu pedido" para ser redirecionamento para o WhatsApp e conversar diretamente com o restaurante.

### Link da API do WhatsApp 📱

O botão de pedido está conectado com a API oficial do WhatsApp. Ao clicar, o cliente será redirecionado para a seguinte URL:

https://api.whatsapp.com/send?phone=0012123456789&text=Oi, vim pelo site! Quero fazer um pedido!

Observações:
- Troque o número de telefone para o número que receberá a mensagem.
[00] Prefixo
[12] DDD
[123456789] Número do relefone do restaurante
- Altere a mensagem de acordo com a necessidade.

## Funcionalidades ✅

- Página única (landing page)
- Layout com interface minimalista, elegante e limpo
- Tipografia personalizada com as fontes:
  - [Spectral](https://fonts.google.com/specimen/Spectral)
  - [Beth Ellen](https://fonts.google.com/specimen/Beth+Ellen)
- Imagem de fundo temática, reforçando o estilo oriental
- **Animações suaves** de entrada nos elementos da página
- **Integração com API do WhatsApp**: o botão "Fazer meu pedido" redireciona diretamente para o WhatsApp do restaurante
- **Responsividade**: adaptada para dispositivos móveis

---

## Tecnologias Utilizadas ⚙️

- HTML5
- CSS3
- Animações via `@keyframes`
- Responsividade com `media queries`, garantindo uma boa experiência tanto em computadores quanto em smartphones.
- Integração com **API do WhatsApp**
